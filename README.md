# Programmering 1 - Turtle Template

En samling uppgifter för att lära sig grund i programmering med Python och turtle-biblioteket.

## Installation

### Förutsättningar
- Python 3.x installerat på datorn

### Setup
1. Klona eller ladda ned detta repo
2. Öppna en terminal och navigera till projektmappen
3. Python kommer redan med turtle-biblioteket, så ingen installation behövs!

## Hur man kör uppgifterna

Varje uppgift är en separat Python-fil (`.py`). Du kan köra dem på följande sätt:

```bash
python uppg1_a_turtle_is_born.py
```

Byt `uppg1_a_turtle_is_born.py` till den uppgift du vill köra.

## Uppgifter

Jobba igenom uppgifterna i ordning:

1. **uppg1_a_turtle_is_born.py** - Introducering av turtle och cirklar
2. **uppg2_kvadrat.py** - Rita kvadrater och positionering
3. **uppg3_smiley.py** - Använd variabler för att skapa en smiley
4. **uppg4_crazy_pattern.py** - Skapa mönster med for-loopar
5. **uppg5_n_star.py** - Rita stjärnor med udda antal hörn
6. **uppg6_n_star_2.py** - Rita stjärnor med jämnt antal hörn
7. **uppg7_clock.py** - Designa en klocka
8. **uppg8_race.py** - Två turtles som tävlar

## Kommando-referens

Se [kommandon.md](kommandon.md) för en komplett lista över alla turtle-kommandon.

## Tips

- Läs uppgiftsbeskrivningen noggrant innan du börjar
- Experimentera! Prova att ändra värden och se vad som händer
- Använd `turtle.done()` för att behålla fönstret öppet när programmet är klart
- Du kan stänga fönstret genom att klicka på X-knappen

Lycka till! 🐢
